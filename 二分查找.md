### LeetCode704

```java
class Solution {
    public int search(int[] nums, int target) {
        int left=0,right=nums.length-1;
        while(left<=right){
            int mid=(left+right)/2;
            if(nums[mid]==target){
                return mid;
            }else if(nums[mid]>target){
                right=mid-1;
            }else{
                left=mid+1;
            }
        }
        return -1;
    }
}
```

### LeetCode 50快速幂

```java
class Solution{
    public double pow(double x,int n){
        long N=n;
        if(n<0){
           x=1/x;
           N=-N;
        }
        
        double ans=1;
        double current=x;
        while(N>0){
           if((N&1)==1){
               ans=ans*current;
           }
            current=current*current;
            N>>=1;
        }
        return ans;
    }
}
```
